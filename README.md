# Quotes Recommendation Chatbot using NLP

## Project Overview
This project implements a chatbot that recommends quotes based on user requests using Natural Language Processing (NLP).  
The chatbot understands user intents such as motivation, love, or success and returns a relevant quote.

This project is developed as part of the **SmartBridge Program**.

---

## Features
- NLP-based chatbot using Rasa
- Recognizes user intents
- Provides motivational, love, and success quotes
- Uses structured training data
- Simple conversational interface

---

## Technologies Used
- Python
- Rasa Open Source
- Natural Language Processing (NLP)
- Anaconda
- Visual Studio Code
- GitHub
